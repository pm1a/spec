Detailed Description

V/V nastroje.

Matice:

format je nezavisly na typu matice tisknou se jen nenulove prvky

  • prvni radek: pocet_radku pocet_nenul
  • dalsi radky: radek sloupec hodnota

Vektor:

  • prvni radek: pocet_prvku
  • dalsi radky: hodnota
